# ATRS Licensing

ATRS uses a dual-license structure:

- Code under `src/`, tests, fixtures, and package build files are licensed under the MIT License.
- Schemas, governance files, RFCs, and documentation are available under Creative Commons Attribution 4.0 International (CC-BY-4.0).

The generated public repository also includes an MIT `LICENSE` file for package manager and GitHub compatibility. This file records the document/schema licensing split.
